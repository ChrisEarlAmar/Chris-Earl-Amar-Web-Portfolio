CEA-HBW

CEA-HBW is a hotel booking website project developed using PHP and MySQL. The website provides a simulated environment for hotel booking, assuming payment has been made externally. It currently features a customer UI displaying the site's information and an admin UI with backend functionality. Future updates aim to include user registration and login for customers, booking and reservation management, as well as profile management and hotel reviews.

Languages & Technologies
PHP JavaScript Ajax MySQL Bootstrap CSS
CEA-SBW

CEA-SBW is a Laravel simple blogging website that incorporates the Unsplash Image Search API for adding images to blog posts. It features user authentication, allowing users to register, log in, and create blog posts with titles, short descriptions, content, and images fetched from Unsplash. Users can manage their blogs by deleting them. Future enhancements include enabling users to manage and edit their profiles, select saved Unsplash images for blog posts, and edit existing blog posts.

Languages & Technologies
PHP Unsplash API Laravel MySQL Bootstrap CSS
